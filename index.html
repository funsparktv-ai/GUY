GUY Bot Commands

Your friendly Discord bot commands, organized by category!

General Commands

  • ?ping - Check bot's latency.
  • ?say <message> - Make the bot say something.
  • ?avatar [member] - Get the avatar of a member.
  • ?userinfo [member] ⭐ - Get information about a member.
  • ?serverinfo ⭐ - Get information about the server.

Moderation Commands

  • ?kick <member> [reason] - Kick a member from the server.
  • ?ban <member> [reason] - Ban a member from the server.
  • ?unban <username#discriminator> - Unban a banned member.
  • ?clear [amount] - Clear messages in the channel.
  • ?purge [amount] ⭐ - Purges a specified number of messages from the channel.
  • ?warn [user] [reason] - Warn a user.
  • ?clearwarnings [user] - Clear warnings of a user.
  • ?mute [user] - Mute a user.
  • ?unmute [user] - Unmute a user.

Fun and Games Commands

  • ?roll <number> - Roll a dice with the specified number of sides.
  • ?e - Send a random 'E' gif.
  • ?addquote <text> - Add a Quote for all servers!
  • ?quote - Get a random quote!
  • ?weather <city> - Get the current weather of a city.
  • ?set_birthday <date> - Set your birthday.
  • ?birthdays_today - Check if there are any birthdays today.
  • ?joke - Get a random joke!
  • ?meme - Get a random meme!
  • ?pug ⭐ - Get a cute pug picture.
  • ?rps <choice> ⭐ - Play rock, paper, scissors with the bot.
  • ?cat ⭐ - Get a cute cat picture.
  • ?dadjoke ⭐ - Get a random dad joke.
  • ?dog ⭐ - Get a cute dog picture.
  • ?flip ⭐ - Flip a coin.
